OnioGo - Advanced Tor controller






Control and Monitor the Tor software with User-friendly interface




Main features:

  • Simple interface for easy setup and configuration.

  • Preconfigured or easy configurable (see "Quick start" page)

  • Multi-threaded control client

  • Native desktop application for Windows and Linux 64-bit systems (32 bit eventually for request)

  • Portable

  • Great and unique method of graphical representation of circuits and streams relation (included history of relations)

  • Many control protocol functions accessible directly from context and application menu

  • Circuits window with many information about circuits with building progress bar

  • Streams window with many information about streams

  • Drag and drop operations between Circuit and Stream windows, possibility drag "unattached" or "detached"streams and drop to the desired circuit

  • Close stuck circuits and streams just by mouse click

  • Speeds and transferred data for every circuit and stream

  • Traffic window with automatic Y axis and transferred data for every minute

  • Commands window with many controls, circuit and stream build-in function, terminal - like command row

  • Map of the world with circuit visualization

  • Router list with much available info and rich popup menu with many functions

  • Onion services window *

  • Tor server window *

  • Fast processing with "network status", "extra info status" and "country codes" caches with fast synchronization

  • Unique Scheduler for processing many build-in functions or any function from control protocol

  • Frequently updates and functions on request eventually

  • Long history, real project, not student or test attempt

  • Tiny executable size

  • And many more.
